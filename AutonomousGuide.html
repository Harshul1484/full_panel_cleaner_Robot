HOW THE AUTONOMOUS SYSTEM WORKS
Concepts before you build anything

The Core Idea

The robot cleans a solar panel by driving in an S-pattern (boustrophedon sweep) — straight across, turn, straight back, turn, repeat — until the whole panel is covered. It uses sensors to detect panel edges so it knows when to turn, and obstacle sensors to stop if it hits something unexpected.

ADDITIONAL COMPONENTS FOR FULL AUTONOMY
What to buy on top of the base build

Why Each Sensor is Needed

IR Edge Sensors × 4

Detect the edge of the solar panel. When the sensor stops seeing a surface below it, the robot knows it has reached the end of a row and must turn. One at each corner.

HC-SR04 Ultrasonic × 2

Detect obstacles in front (e.g. panel frame, bird, debris). If something is closer than 8cm the robot stops and waits. Two sensors give left/right coverage.

DS3231 RTC Module

Real-time clock. Keeps accurate time even when powered off. You set a daily alarm (e.g. 6:00 AM) and the robot automatically starts cleaning without any human input.

ESP8266 WiFi Module

Reports robot status (cleaning / done / low battery / error) to your phone via a simple web page or MQTT. You can monitor from anywhere on the same WiFi network.

IR Dock Sensor × 1

A simple IR LED + photodiode pair on the docking station. When the robot returns and the dock sensor triggers, it knows it has arrived and shuts down.

MPU-6050 IMU (already listed)

Detects if the robot tilts too much (panel angle changes, slip). Triggers E-STOP if tilt exceeds threshold. Already in base BOM — confirm it is installed.

Exact positions for every sensor on the robot body

Sensor Mounting Positions

Step-by-Step Sensor Installation

STEP 1
Mount IR Edge Sensors (× 4 corners)
  • Use TCRT5000 IR sensor modules — they have VCC, GND, DO (digital out), AO (analog out)
  • Mount one sensor pointing straight down at each of the 4 corners of the robot chassis
  • Sensor face must be 5–15mm above the solar panel surface when robot is on the panel
  • Use M2 screws or hot glue into a small 3D-printed bracket at each corner
  • Angle sensor slightly outward (10–15°) so it detects the edge before the wheel reaches it
  • Route sensor wires along the inside of the chassis frame — use cable clips
  • Label each wire: FL, FR, RL, RR
✓ Test each sensor with a multimeter before installing. Hold sensor over a white surface → DO pin should be LOW (0V). Move off surface (into air) → DO pin should go HIGH (5V).
STEP 2
Mount HC-SR04 Ultrasonic Sensors (× 2 front)
  • Mount both sensors on the FRONT face of the robot, side by side
  • Left sensor: centered on left half of front face
  • Right sensor: centered on right half of front face
  • Sensors must face forward horizontally — parallel to the panel surface
  • Use M3 screws through sensor mounting holes into the chassis front plate
  • Keep sensors at least 3cm apart to avoid acoustic interference
  • Route trigger + echo wires back through the chassis to MCU
⚠ HC-SR04 runs on 5V but ECHO pin outputs 5V which can damage Arduino input if not using 5V-tolerant pins. Add a voltage divider on ECHO: 1kΩ series + 2kΩ to GND → gives 3.3V safe signal. Or use HC-SR04P (3.3V version).
STEP 3
Mount DS3231 RTC Module
  • Mount inside chassis on electronics plate using M2 nylon standoffs
  • Insert CR2032 coin cell into the RTC module battery holder before mounting
  • Connect via I2C bus (shared with MPU-6050 — same SDA/SCL lines)
  • DS3231 I2C address is 0x68 — MPU-6050 also defaults to 0x68
  • Set MPU-6050 AD0 pin HIGH to change its address to 0x69 — avoids conflict
  • Connect a small wire from AD0 on MPU-6050 to 3.3V/5V rail to permanently set to 0x69
ℹ I2C bus can have multiple devices — DS3231 (0x68) and MPU-6050 (0x69) share the same SDA/SCL wires. Only two wires needed for both devices.
STEP 4
Mount ESP8266 NodeMCU (WiFi Module)
  • Mount on electronics plate using M3 nylon standoffs
  • Power from 3.3V rail (ESP8266 is 3.3V — do NOT connect to 5V directly)
  • Connect ESP8266 RX → Arduino Mega TX2 (D16) via voltage divider (5V → 3.3V): 1kΩ + 2kΩ
  • Connect ESP8266 TX → Arduino Mega RX2 (D17) directly (3.3V is safe for Mega input)
  • ESP8266 GND → Common GND
  • Ensure antenna end of ESP8266 is near the robot edge — not buried under metal
STEP 5
Mount Dock IR Sensor Pair
  • IR LED: mount on the docking station, pointing outward
  • IR photodiode/receiver: mount on the REAR of the robot
  • When robot backs into dock, IR beam hits receiver → dock detected
  • Wire IR LED: 5V → 220Ω resistor → LED+ → LED− → GND
  • Wire IR receiver: VCC → 5V, GND → GND, OUT → Arduino D30
  • Add 10kΩ pull-up resistor from OUT to 5V on receiver
  • Align LED and receiver at same height — test beam alignment before final mount

#include <Wire.h>
#include <RTClib.h>
#include <MPU6050.h>

// ─── PIN DEFINITIONS ──────────────────────────────────────────
// BTS7960 #1 — LEFT GF555 motor (left track)
#define L_RPWM      9    // left motor forward PWM
#define L_LPWM     10    // left motor reverse PWM
#define L_REN       2    // left motor forward enable
#define L_LEN       3    // left motor reverse enable

// BTS7960 #2 — RIGHT GF555 motor (right track)
#define R_RPWM     11    // right motor forward PWM
#define R_LPWM     12    // right motor reverse PWM
#define R_REN       4    // right motor forward enable
#define R_LEN       5    // right motor reverse enable

// Brush motors (L298N) — drives both 6331K61 brush rollers
#define PIN_BRUSH1_EN  6
#define PIN_BRUSH2_EN  7
#define PIN_BRUSH1_IN1 22
#define PIN_BRUSH1_IN2 23
#define PIN_BRUSH2_IN3 24
#define PIN_BRUSH2_IN4 25

// Actuators
#define PIN_PUMP      8
#define PIN_SOLENOID 13

// Encoders — hardware interrupt pins only (D18/D19 = INT5/INT4 on Mega)
#define PIN_ENC_L_A  18   // left GF555 encoder channel A
#define PIN_ENC_L_B  19   // left GF555 encoder channel B
#define PIN_ENC_R_A  36   // right GF555 encoder channel A (NOT I2C pins!)
#define PIN_ENC_R_B  37   // right GF555 encoder channel B

// IR edge sensors (LOW = on panel, HIGH = off panel/edge)
#define PIN_IR_FL   26
#define PIN_IR_FR   27
#define PIN_IR_RL   28
#define PIN_IR_RR   29

// Ultrasonic (front obstacle)
#define PIN_US_L_TRIG 30
#define PIN_US_L_ECHO 31
#define PIN_US_R_TRIG 32
#define PIN_US_R_ECHO 33

// Dock sensor
#define PIN_DOCK    34

// RTC alarm interrupt — D20 conflicts with I2C SDA, use D35 instead with PCINT
#define PIN_RTC_INT  35

// Analog
#define PIN_WATER_LVL A0
#define PIN_VBAT      A1

// ─── CONSTANTS ────────────────────────────────────────────────
#define DRIVE_SPEED_FWD   180   // 0-255 PWM, ~70% speed
#define DRIVE_SPEED_TURN  120   // turn speed for differential steering
#define DRIVE_SPEED_DOCK  100   // slow approach to dock
#define BRUSH_SPEED       200   // brush motor PWM
#define OBSTACLE_CM         8   // stop if object closer than 8cm
#define SPIN_TIME_MS     1800   // ms for 180° in-place spin (calibrate!)
#define SHIFT_TIME_MS     700   // ms to shift one row width after spin
#define MAX_ROWS           14   // panel 2000mm / brush ~143mm = 14 rows
#define LOW_BATTERY_MV  21000   // return to dock below 21V (24V pack)
#define LOW_WATER_VAL     100   // ADC value below which water is low
#define TILT_LIMIT_DEG     20   // E-STOP if tilt exceeds 20°

// ─── STATE MACHINE ────────────────────────────────────────────
enum State {
  STATE_IDLE,
  STATE_START,
  STATE_SWEEP,
  STATE_TURN,
  STATE_OBSTACLE,
  STATE_DONE,
  STATE_DOCK,
  STATE_ESTOP
};

State currentState = STATE_IDLE;
int  rowCount = 0;
bool sweepDirection = true;  // true = forward, false = reverse
volatile bool rtcAlarm = false;

// ─── OBJECTS ──────────────────────────────────────────────────
RTC_DS3231 rtc;
MPU6050    mpu;

// ─── ENCODERS — one per GF555 motor ───────────────────────────
volatile long encLeft  = 0;
volatile long encRight = 0;
void encLeftISR()  { encLeft++;  }
void encRightISR() { encRight++; }

// ─── RTC ALARM ISR ────────────────────────────────────────────
void rtcAlarmISR() { rtcAlarm = true; }

// ═══════════════════════════════════════════════════════════════
// SETUP
// ═══════════════════════════════════════════════════════════════
void setup() {
  Serial.begin(115200);
  Serial2.begin(115200);  // to ESP8266
  Wire.begin();

  // BTS7960 #1 — LEFT motor
  pinMode(L_RPWM, OUTPUT);  pinMode(L_LPWM, OUTPUT);
  pinMode(L_REN,  OUTPUT);  pinMode(L_LEN,  OUTPUT);

  // BTS7960 #2 — RIGHT motor
  pinMode(R_RPWM, OUTPUT);  pinMode(R_LPWM, OUTPUT);
  pinMode(R_REN,  OUTPUT);  pinMode(R_LEN,  OUTPUT);

  // Brush pins
  pinMode(PIN_BRUSH1_EN,  OUTPUT);
  pinMode(PIN_BRUSH2_EN,  OUTPUT);
  pinMode(PIN_BRUSH1_IN1, OUTPUT);
  pinMode(PIN_BRUSH1_IN2, OUTPUT);
  pinMode(PIN_BRUSH2_IN3, OUTPUT);
  pinMode(PIN_BRUSH2_IN4, OUTPUT);

  // Actuators
  pinMode(PIN_PUMP,     OUTPUT);
  pinMode(PIN_SOLENOID, OUTPUT);

  // Sensors
  pinMode(PIN_IR_FL, INPUT);  pinMode(PIN_IR_FR, INPUT);
  pinMode(PIN_IR_RL, INPUT);  pinMode(PIN_IR_RR, INPUT);
  pinMode(PIN_US_L_TRIG, OUTPUT);  pinMode(PIN_US_L_ECHO, INPUT);
  pinMode(PIN_US_R_TRIG, OUTPUT);  pinMode(PIN_US_R_ECHO, INPUT);
  pinMode(PIN_DOCK, INPUT_PULLUP);

  // Encoder interrupts — D18=INT5, D19=INT4 on Mega (hardware ISR)
  attachInterrupt(digitalPinToInterrupt(PIN_ENC_L_A), encLeftISR,  RISING);
  attachInterrupt(digitalPinToInterrupt(PIN_ENC_L_B), encRightISR, RISING); // D19=INT4

  // RTC setup
  if (!rtc.begin()) {
    Serial.println("RTC not found!");
  } else {
    rtc.disable32K();
    rtc.clearAlarm(1);
    rtc.clearAlarm(2);
    // Set daily alarm at 06:00:00
    rtc.setAlarm1(DateTime(0,0,0,6,0,0), DS3231_A1_Hour);
    pinMode(PIN_RTC_INT, INPUT_PULLUP);
    // D35 uses PCINT2 (pin change interrupt, not hardware INT)
    // Simplest: poll rtc.alarmFired() in loop() instead of using interrupt
    // attachInterrupt not available on D35 — use digitalRead polling below
  }

  // IMU setup
  mpu.initialize();
  mpu.setFullScaleGyroRange(MPU6050_GYRO_FS_250);
  mpu.setFullScaleAccelRange(MPU6050_ACCEL_FS_2);

  // All off at start
  driveStop();
  brushOff();
  waterOff();

  Serial.println("SolarBot READY — waiting for RTC alarm (06:00)");
  wifiSend("STATUS:IDLE");
}

// ═══════════════════════════════════════════════════════════════
// MAIN LOOP — STATE MACHINE
// ═══════════════════════════════════════════════════════════════
void loop() {

  // Always check E-STOP conditions
  if (checkEStop() && currentState != STATE_ESTOP) {
    currentState = STATE_ESTOP;
  }

  switch (currentState) {

    case STATE_IDLE:
      // Poll RTC alarm (D35 is PCINT, not hardware INT)
      if (rtcAlarm || rtc.alarmFired(1)) {
        rtcAlarm = false;
        rtc.clearAlarm(1);
        currentState = STATE_START;
        Serial.println("RTC alarm → starting clean");
        wifiSend("STATUS:STARTING");
      }
      break;

    case STATE_START:
      rowCount = 0;
      sweepDirection = true;
      brushOn();
      waterOn();
      delay(500);           // warm-up
      currentState = STATE_SWEEP;
      wifiSend("STATUS:SWEEPING");
      break;

    case STATE_SWEEP:
      driveSweep();
      break;

    case STATE_TURN:
      doTurn();
      break;

    case STATE_OBSTACLE:
      handleObstacle();
      break;

    case STATE_DONE:
      brushOff();
      waterOff();
      driveToDock();
      break;

    case STATE_DOCK:
      driveStop();
      brushOff();
      waterOff();
      wifiSend("STATUS:DOCKED_CHARGING");
      delay(60000);          // wait 1 minute then go idle
      currentState = STATE_IDLE;
      break;

    case STATE_ESTOP:
      driveStop();
      brushOff();
      waterOff();
      wifiSend("STATUS:ESTOP");
      delay(5000);
      break;
  }
}

// ═══════════════════════════════════════════════════════════════
// DRIVE FUNCTIONS — DIFFERENTIAL STEERING (2× GF555 + 2× BTS7960)
// Left track = BTS7960 #1 (L_RPWM/L_LPWM/L_REN/L_LEN)
// Right track = BTS7960 #2 (R_RPWM/R_LPWM/R_REN/R_LEN)
// ═══════════════════════════════════════════════════════════════

// ─── Low-level single-motor helpers ───────────────────────────
void motorFwd(int rpwm, int lpwm, int ren, int len, int spd) {
  digitalWrite(ren, HIGH); digitalWrite(len, LOW);
  analogWrite(rpwm, spd);  analogWrite(lpwm, 0);
}
void motorRev(int rpwm, int lpwm, int ren, int len, int spd) {
  digitalWrite(ren, LOW);  digitalWrite(len, HIGH);
  analogWrite(rpwm, 0);    analogWrite(lpwm, spd);
}
void motorStop(int rpwm, int lpwm, int ren, int len) {
  analogWrite(rpwm, 0);    analogWrite(lpwm, 0);
  digitalWrite(ren, LOW);  digitalWrite(len, LOW);
}

// ─── High-level drive commands ─────────────────────────────────
void driveForward(int spd) {
  // Both tracks forward at same speed → straight ahead
  motorFwd(L_RPWM, L_LPWM, L_REN, L_LEN, spd);
  motorFwd(R_RPWM, R_LPWM, R_REN, R_LEN, spd);
}

void driveReverse(int spd) {
  // Both tracks reverse at same speed → straight back
  motorRev(L_RPWM, L_LPWM, L_REN, L_LEN, spd);
  motorRev(R_RPWM, R_LPWM, R_REN, R_LEN, spd);
}

void driveStop() {
  motorStop(L_RPWM, L_LPWM, L_REN, L_LEN);
  motorStop(R_RPWM, R_LPWM, R_REN, R_LEN);
}

// Spin in place — left FWD + right REV = rotate clockwise (turning left)
void spinLeft(int spd) {
  motorFwd(L_RPWM, L_LPWM, L_REN, L_LEN, spd);
  motorRev(R_RPWM, R_LPWM, R_REN, R_LEN, spd);
}

// Spin in place — right FWD + left REV = rotate counter-clockwise (turning right)
void spinRight(int spd) {
  motorRev(L_RPWM, L_LPWM, L_REN, L_LEN, spd);
  motorFwd(R_RPWM, R_LPWM, R_REN, R_LEN, spd);
}

void driveSweep() {
  // Check for obstacle first
  float distL = measureDistance(PIN_US_L_TRIG, PIN_US_L_ECHO);
  float distR = measureDistance(PIN_US_R_TRIG, PIN_US_R_ECHO);
  if (distL < OBSTACLE_CM || distR < OBSTACLE_CM) {
    currentState = STATE_OBSTACLE;
    return;
  }

  // Check leading-edge sensors based on current sweep direction
  if (sweepDirection) {
    bool edgeFL = digitalRead(PIN_IR_FL);
    bool edgeFR = digitalRead(PIN_IR_FR);
    if (edgeFL || edgeFR) {
      driveStop();
      rowCount++;
      currentState = (rowCount >= MAX_ROWS) ? STATE_DONE : STATE_TURN;
    } else {
      driveForward(DRIVE_SPEED_FWD);
    }
  } else {
    bool edgeRL = digitalRead(PIN_IR_RL);
    bool edgeRR = digitalRead(PIN_IR_RR);
    if (edgeRL || edgeRR) {
      driveStop();
      rowCount++;
      currentState = (rowCount >= MAX_ROWS) ? STATE_DONE : STATE_TURN;
    } else {
      driveReverse(DRIVE_SPEED_FWD);
    }
  }
}

void doTurn() {
  // Differential turn sequence for tracked robot:
  // Step 1: spin 90° in place (half-turn toward new row)
  // Step 2: drive forward one row width (shift)
  // Step 3: spin 90° back to align with sweep direction
  // This gives a clean U-turn without sliding tracks sideways

  driveStop();  delay(200);

  long spinMs = SPIN_TIME_MS / 2;   // 90° = half of 180° time
  spinLeft(DRIVE_SPEED_TURN);        // rotate 90° toward new row
  delay(spinMs);
  driveStop();  delay(150);

  driveForward(DRIVE_SPEED_TURN);    // advance one row width
  delay(SHIFT_TIME_MS);
  driveStop();  delay(150);

  spinLeft(DRIVE_SPEED_TURN);        // rotate another 90° to face new direction
  delay(spinMs);
  driveStop();  delay(200);

  sweepDirection = !sweepDirection;  // flip sweep direction
  currentState = STATE_SWEEP;
}

void driveToDock() {
  if (digitalRead(PIN_DOCK) == LOW) {  // dock IR sensor active low
    driveStop();
    currentState = STATE_DOCK;
    wifiSend("STATUS:DOCKED");
  } else {
    driveReverse(DRIVE_SPEED_DOCK);
  }
}

// BRUSH + WATER
// ═══════════════════════════════════════════════════════════════
void brushOn() {
  digitalWrite(PIN_BRUSH1_IN1, HIGH);
  digitalWrite(PIN_BRUSH1_IN2, LOW);
  digitalWrite(PIN_BRUSH2_IN3, HIGH);
  digitalWrite(PIN_BRUSH2_IN4, LOW);
  analogWrite(PIN_BRUSH1_EN, BRUSH_SPEED);
  analogWrite(PIN_BRUSH2_EN, BRUSH_SPEED);
}

void brushOff() {
  analogWrite(PIN_BRUSH1_EN, 0);
  analogWrite(PIN_BRUSH2_EN, 0);
  digitalWrite(PIN_BRUSH1_IN1, LOW);
  digitalWrite(PIN_BRUSH1_IN2, LOW);
  digitalWrite(PIN_BRUSH2_IN3, LOW);
  digitalWrite(PIN_BRUSH2_IN4, LOW);
}

void waterOn() {
  digitalWrite(PIN_SOLENOID, HIGH);  // open valve
  delay(200);
  digitalWrite(PIN_PUMP, HIGH);      // start pump
}

void waterOff() {
  digitalWrite(PIN_PUMP, LOW);
  delay(300);
  digitalWrite(PIN_SOLENOID, LOW);   // close valve after pump stops
}

// ═══════════════════════════════════════════════════════════════
// SENSORS
// ═══════════════════════════════════════════════════════════════
float measureDistance(int trigPin, int echoPin) {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  long dur = pulseIn(echoPin, HIGH, 30000);  // 30ms timeout
  return dur * 0.0343 / 2.0;  // cm
}

float readTilt() {
  int16_t ax, ay, az, gx, gy, gz;
  mpu.getMotion6(&ax, &ay, &az, &gx, &gy, &gz);
  float tiltX = atan2(ay, az) * 180.0 / PI;
  return abs(tiltX);
}

int readBatteryMV() {
  int raw = analogRead(PIN_VBAT);
  // Voltage divider: R1=100k, R2=10k → scale factor = 11
  return (raw * 5000L / 1023) * 11;
}

bool checkEStop() {
  if (readTilt() > TILT_LIMIT_DEG) {
    Serial.println("ESTOP: tilt exceeded");
    return true;
  }
  if (readBatteryMV() < LOW_BATTERY_MV) {
    Serial.println("ESTOP: low battery");
    wifiSend("STATUS:LOW_BATTERY");
    return true;
  }
  if (analogRead(PIN_WATER_LVL) < LOW_WATER_VAL) {
    Serial.println("ESTOP: no water");
    wifiSend("STATUS:NO_WATER");
    waterOff();
    // Don't full E-STOP for no water — just stop pump, continue dry
    return false;
  }
  return false;
}

void handleObstacle() {
  driveStop();
  wifiSend("STATUS:OBSTACLE");
  delay(1000);

  float distL = measureDistance(PIN_US_L_TRIG, PIN_US_L_ECHO);
  float distR = measureDistance(PIN_US_R_TRIG, PIN_US_R_ECHO);

  if (distL >= OBSTACLE_CM && distR >= OBSTACLE_CM) {
    currentState = STATE_SWEEP;  // obstacle gone, resume
    wifiSend("STATUS:SWEEPING");
  }
  // else stay in OBSTACLE state and retry next loop
}

  Serial2.println(msg);
  Serial.println(msg);
}

ESP8266 WiFi Firmware — esp_wifi.ino

ℹ Upload this separately to the ESP8266 NodeMCU using Arduino IDE. Change SSID and password to your WiFi network.
#include <ESP8266WiFi.h>
#include <ESP8266WebServer.h>

const char* ssid     = "YOUR_WIFI_SSID";
const char* password = "YOUR_WIFI_PASS";

ESP8266WebServer server(80);
String botStatus = "IDLE";

void setup() {
  Serial.begin(115200);  // from Arduino Mega TX2
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) delay(500);

  server.on("/", []() {
    String html = "<h1>SolarBot Status</h1><p>" + botStatus + "</p>";
    html += "<meta http-equiv='refresh' content='5'>";
    server.send(200, "text/html", html);
  });
  server.begin();
}

void loop() {
  if (Serial.available()) {
    botStatus = Serial.readStringUntil('\n');
    botStatus.trim();
  }
  server.handleClient();
}

CALIBRATION
Must do before first autonomous run
CALIBRATION 1
IR Edge Sensor Threshold
  • Place robot ON the solar panel surface
  • Open Serial Monitor → read IR sensor DO pin value (should be LOW = 0)
  • Slowly slide robot to the edge — watch when DO flips HIGH
  • Adjust the small potentiometer on TCRT5000 module until it triggers 1–2cm before the wheel reaches the edge
  • Do this for all 4 sensors
  • Write down the physical gap at which each triggers — should be consistent
CALIBRATION 2
SPIN_TIME_MS and SHIFT_TIME_MS (Differential Turn)
  • The robot turns using differential drive: left motor FWD + right motor REV = spin in place
  • Place robot on a flat surface with a tape cross marking the centre point
  • Run a test sketch that calls spinLeft(DRIVE_SPEED_TURN); delay(SPIN_TIME_MS); driveStop();
  • Measure the rotation angle — adjust SPIN_TIME_MS until robot rotates exactly 90°
  • Typical SPIN_TIME_MS for 90°: 900–1200ms (on solar panel glass, less friction than floor)
  • Then test doTurn() — robot should spin 90°, drive forward one row, spin 90° again
  • Measure lateral shift — adjust SHIFT_TIME_MS until shift = exactly one brush roller width (~143mm)
  • Typical SHIFT_TIME_MS: 600–900ms
⚠ Calibrate on the actual solar panel glass surface — friction is significantly less than a floor and will affect both SPIN_TIME_MS and SHIFT_TIME_MS.
CALIBRATION 3
MAX_ROWS Count
  • Measure panel width (e.g. 1650mm) and brush roller width (e.g. 150mm)
  • MAX_ROWS = ceil(panel_width / brush_width) = ceil(1650/150) = 11
  • Set MAX_ROWS in firmware accordingly
  • Add 1 extra row as buffer to ensure full coverage
CALIBRATION 4
RTC Time Set
  • Upload a one-time sketch to set the RTC time to current time:
#include <Wire.h>
#include <RTClib.h>
RTC_DS3231 rtc;
void setup() {
  Wire.begin();
  rtc.begin();
  // Set to compile time automatically:
  rtc.adjust(DateTime(F(__DATE__), F(__TIME__)));
}
void loop() {}
  • Upload this, wait 2 seconds, then upload the main autonomous firmware
  • RTC will now keep accurate time even when robot is powered off
  • Change the alarm time in setup() if you want cleaning at a different hour
CALIBRATION 5
Battery Voltage Calibration
  • Measure actual battery voltage with multimeter → e.g. 23.8V
  • Read raw ADC value from Serial Monitor → e.g. 434
  • Calculate: scale = 23800 / (434 * 5000/1023) = actual_mv / adc_mv
  • Adjust the *11 multiplier in readBatteryMV() to match your divider resistors
  • Set LOW_BATTERY_MV to ~80% of full charge voltage
